📖 How to Use This JSON Formatter

  1. Paste your JSON into the Input area or drag & drop a .json file. The tool instantly validates it and shows ✓ Valid or ✗ Invalid.
  2. Click Beautify to format with proper indentation, or Minify to compress into a single line for API payloads.
  3. Sort Keys alphabetically for consistent ordering — great for comparing configs or debugging diffs.
  4. Convert to CSV — click 📊 CSV to transform arrays of objects into downloadable CSV. Nested objects are auto-flattened with dot notation.
  5. Convert to YAML — click 📄 YAML to get clean, indented YAML output. Perfect for Kubernetes configs, Docker Compose, or CI/CD files.
  6. Tree View — click 🌳 Tree for an interactive, collapsible tree. Expand/collapse nodes to explore deep JSON structures.
  7. History — click 📜 History to see your last 8 formatted JSONs. One-click restore any previous entry.
  8. Copy or Share — use 📋 Copy to grab the output, or 🔗 Share to generate a URL with your JSON pre-loaded (no server storage).

❓ Frequently Asked Questions

Is this JSON formatter safe to use with sensitive data?

Yes. JSONForge runs 100% in your browser. Your JSON data is never sent to any server. You can verify this by opening your browser's Network tab — zero requests are made when you paste or format JSON.

What is the maximum JSON file size I can format?

JSONForge can handle JSON files up to several megabytes directly in your browser. Performance depends on your device, but typical API responses and config files are processed instantly.

How do I validate JSON syntax?

Simply paste your JSON into the input area. JSONForge automatically validates the syntax and shows a green checkmark for valid JSON, or a red error message with the exact line and position of any syntax error.

Can I minify JSON to reduce file size?

Yes. Click the Minify button to remove all whitespace and newlines from your JSON, producing the smallest possible output. This is useful for API payloads, config files, and reducing bandwidth.

How do I convert JSON to YAML?

Click the 📄 YAML button in the toolbar. JSONForge converts your JSON to clean, properly indented YAML format. Arrays use dash notation, nested objects are indented, and special characters are automatically quoted. Perfect for Kubernetes, Docker Compose, and CI/CD configs. No external libraries needed.

Can I drag and drop a JSON file?

Yes! Simply drag any .json file from your computer and drop it onto the input area. JSONForge reads the file locally — nothing is uploaded to any server. You can also click the 📂 File button to open a file picker.

Does it save my formatting history?

Yes. JSONForge saves your last 8 formatted JSONs in your browser's localStorage. Click 📜 History to see recent entries with timestamps, preview the first line, and restore any previous JSON with one click. History never leaves your device.

How do I sort JSON keys alphabetically?

Click the Sort Keys button. JSONForge recursively sorts all object keys in alphabetical order, making it easy to compare JSON objects or maintain consistent key ordering in config files.

Can I share my formatted JSON with someone?

Yes! Click the 🔗 Share button to generate a URL that contains your JSON data. Send this link to anyone — they'll see your exact JSON pre-loaded and formatted. No server storage needed.

How do I convert JSON to CSV?

Paste an array of JSON objects into the input area, then click the 📊 CSV button. JSONForge automatically flattens nested objects using dot notation (e.g., address.city), detects all columns, and generates a downloadable CSV file.

Can I view JSON as a collapsible tree?

Yes! Click the 🌳 Tree button to see a color-coded, collapsible tree visualization. Keys are purple, strings green, numbers amber, booleans blue, and null red. Click ▼/▶ arrows to expand or collapse any node.
