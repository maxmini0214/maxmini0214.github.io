CSS Gradient Syntax Quick Reference

Linear: background: linear-gradient(135deg, #00ff88, #00d4ff);

Radial: background: radial-gradient(circle, #00ff88, #00d4ff);

With stops: background: linear-gradient(to right, red 0%, blue 50%, green 100%);

Text gradient:

background: linear-gradient(135deg, #00ff88, #00d4ff);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;

❓ Frequently Asked Questions

What's the difference between linear-gradient and radial-gradient?

linear-gradient transitions along a line (left→right, top→bottom, or any angle). radial-gradient radiates from a center point outward. Use linear for page backgrounds, headers, and cards. Use radial for spotlight effects, buttons, and circular highlights.

How do I create a gradient with JavaScript?

Set the background property directly:

element.style.background = `linear-gradient(${angle}deg, ${c1}, ${c2})`;

For React: style={{ background: `linear-gradient(135deg, ${c1}, ${c2})` }}

In Tailwind CSS: bg-gradient-to-r from-green-400 to-blue-500

Can I animate a CSS gradient?

CSS can't directly animate background gradients. Two workarounds:

1. CSS Houdini (modern browsers): Register a custom property with @property and animate it.

2. Background-size trick: Set background-size: 200% 200% and animate background-position with keyframes. This creates a smooth gradient shift effect.

What are the best gradient color combinations?

Popular pairs: Blue→Purple (#667eea→#764ba2), Green→Cyan (#00ff88→#00d4ff), Orange→Pink (#f093fb→#f5576c), Dark→Teal (#0f2027→#2c5364).

Tip: Analogous colors (neighbors on the color wheel) = smooth. Complementary colors = bold contrast. Click 🎲 Random to discover unexpected combos!

How do I create gradient text in CSS?

Use background-clip: text:

h1 {
  background: linear-gradient(135deg, #00ff88, #00d4ff);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
