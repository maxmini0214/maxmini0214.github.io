❓ Frequently Asked Questions

How do I generate a SHA-256 hash?

Type or paste text in the Text tab — hashes generate instantly as you type. For files, use the File Hash tab to drag-and-drop. Everything runs in your browser using Web Crypto API.

Can I hash a file without uploading?

Yes! Files are read locally using the File API and hashed in your browser. Nothing is sent to any server. Works with files of any size — large files show a progress indicator.

How do I verify a file checksum?

Use the Compare tab. Hash your file, then paste the expected checksum. The tool compares them character by character. Useful for verifying ISO downloads, software packages, etc.

What is HMAC?

HMAC (Hash-based Message Authentication Code) combines a hash function with a secret key. It verifies both data integrity and authenticity. Common in API authentication and JWT signing.

MD5 vs SHA-256 — which should I use?

SHA-256 for anything security-related. MD5 is fast but cryptographically broken (vulnerable to collisions). Use MD5 only for legacy compatibility or non-security checksums like file deduplication.

Is my data safe?

100% safe. All computation happens in your browser using Web Crypto API. No server calls. Works completely offline once loaded. Your text and files never leave your device.

Can I compare two hashes?

Yes. The Compare tab lets you paste two hashes and check for exact match (case-insensitive). You can also hash a file in the File tab, then compare in the Compare tab.

Does file hashing work with large files?

Yes. Files are processed in chunks using streaming reads, so even multi-GB files work without running out of memory. A progress bar shows the hashing progress for large files.

📖 How to Use

  1. Text: Type or paste text → hashes auto-generate
  2. File: Drag & drop or click to select → MD5/SHA-1/SHA-256/SHA-512 computed locally
  3. Compare: Paste expected + actual hash → instant match/mismatch
  4. HMAC: Enter message + secret key → HMAC-SHA256/384/512 generated
  5. Copy: Click any hash result to copy to clipboard
  6. History: Your last 8 hashes are saved locally for quick reference
