📖 How to Use This Markdown Editor

  1. Write Markdown in the left pane — headings (#), bold (**text**), lists (- item), code blocks (```), tables, and more.
  2. See live preview instantly in the right pane as you type. No need to click "render".
  3. Use the toolbar for quick formatting — bold, italic, strikethrough, code, links, tables, and task lists.
  4. Import files — click "📂 Import" or drag-and-drop any .md file directly onto the editor.
  5. Export your work — "📋 Copy HTML" for raw markup, "📥 Export HTML" for a standalone styled HTML file, or "💾 Download .md" for the source file.
  6. Print or share — click "🖨️ Print" for a clean printable view, or "🔗 Share" to generate a URL with your content embedded.

❓ Frequently Asked Questions

What is Markdown and why should I use it?

Markdown is a lightweight markup language created by John Gruber in 2004. It uses simple symbols like # for headings, ** for bold, and - for lists. It's used across GitHub, documentation sites, blogs, and note-taking apps. Unlike HTML, Markdown is human-readable even in raw form.

What is GitHub Flavored Markdown (GFM)?

GFM extends standard Markdown with: tables (using | pipes), task lists (- [x] done), strikethrough (~~text~~), fenced code blocks with language hints (```js), and autolinked URLs. This editor supports all GFM features.

How do I create a table in Markdown?

Use pipes and hyphens:

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |

Align columns with colons: |:---| left, |:---:| center, |---:| right.

How do I convert Markdown to HTML programmatically?

Popular libraries:

  • JavaScript: marked.parse('# Hello') (marked.js)
  • Node.js: markdown-it or remark
  • Python: import markdown; markdown.markdown('# Hello')
  • CLI: pandoc input.md -o output.html
Is my data safe in this editor?

Yes. This editor runs 100% in your browser — no data is sent to any server. There's no account, no cloud sync, and no tracking of your content. Your Markdown stays on your machine. Check the network tab in DevTools to verify: zero requests.

Can I share my Markdown document with someone?

Click the 🔗 Share button in the header. It generates a URL with your Markdown content encoded in the URL parameter. Anyone who opens the link sees your exact document loaded and rendered instantly. You can also download the .md file directly.

How do I convert a Markdown file to an HTML file?

Click 📥 Export HTML in the toolbar. It generates a standalone HTML file with embedded CSS styling that matches the preview exactly. The file works offline — open it in any browser. For raw HTML markup (to paste into a CMS), use 📋 Copy HTML instead.

Can I open a .md file from my computer?

Yes! Click 📂 Import or drag and drop any .md, .markdown, or .txt file directly onto the editor. The content loads instantly with live preview. Your file stays in your browser — nothing is uploaded.

Does this editor auto-save my work?

Yes. Your document is auto-saved to localStorage every few seconds. If you close the tab and return, your work is restored. Click 📜 History to see your last 8 saved documents and restore any of them with one click.

How do I print my Markdown document?

Click 🖨️ Print or press Ctrl+P (Cmd+P on Mac). The print view hides the editor and UI, showing only the clean rendered preview optimized for paper. Links display their URLs for offline reference.
