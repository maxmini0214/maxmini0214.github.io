❓ Frequently Asked Questions

What is Lorem Ipsum?▼
Lorem Ipsum is dummy text used in printing and typesetting since the 1500s. It originates from Cicero's "De Finibus Bonorum et Malorum" (45 BC). Designers use it to preview layouts without being distracted by readable content.
How do I generate a specific number of characters?▼
Click the "🎯 Character Limit" tab, enter your target (e.g., 500), and click Generate. The tool produces text as close to your target as possible without cutting words. Quick presets: 140 (Tweet), 160 (SMS), 255 (VARCHAR).
What themed text options are available?▼
Five themes: Classic (traditional Latin), Hipster (artisan, cold-pressed, sustainable), Tech (API, blockchain, microservices), Office (synergy, KPI, stakeholder), and Korean (한국어 더미 텍스트). Each generates natural-sounding placeholder text.
Can I copy as HTML, Markdown, or JSON?▼
Yes! Click "📄 HTML" for <p> wrapped paragraphs, "📝 Markdown" for markdown formatting, or "📦 JSON" for a JSON array. The JSON Output tab also generates objects with metadata and API mock responses.
Is this tool free and private?▼
100% free and private. All text generation runs in your browser with JavaScript. Nothing is sent to any server. Works offline after loading.
What is Hipster Ipsum?▼
Hipster Ipsum uses trendy, artisan vocabulary instead of Latin — words like "cold-pressed", "sustainable", "artisanal", "craft beer". Great for lifestyle, food, and design mockups that need a modern feel.
How do I generate JSON placeholder data?▼
Use the "📦 JSON Output" tab. Choose format: simple array, object with metadata (theme, word count), or API mock response (with id, title, body, timestamps). Perfect for frontend testing and API mocking.
Can I generate Korean placeholder text?▼
Yes! Select the "🇰🇷 Korean" theme. It produces natural Korean dummy text using common vocabulary — perfect for Korean website mockups and app prototyping.
How do I generate fake test data?▼
Click the "🎭 Fake Data" tab. Select fields (name, email, phone, address, UUID, etc.), choose how many rows, and click Generate. Export as JSON, CSV, TSV, or SQL INSERT statements. Great for database seeding and API mocking.
Can I export fake data as SQL INSERT statements?▼
Yes! In the Fake Data tab, select "SQL INSERT" as the export format and set your table name. Each row becomes a valid INSERT statement ready to paste into your database console or migration file.
Is the fake data realistic?▼
The generator uses pools of real-world names, cities, companies, and job titles for both English (US) and Korean locales. Emails are generated from names with realistic domains. All data is randomly generated in your browser — no real personal information is used.
