❓ Frequently Asked Questions

How do I check WCAG color contrast?
Use the ♿ Contrast Checker tab. Enter foreground and background colors. The tool calculates the contrast ratio and shows pass/fail for WCAG 2.1 AA (4.5:1 normal, 3:1 large) and AAA (7:1 normal, 4.5:1 large).
What contrast ratio is accessible?
WCAG 2.1 Level AA requires 4.5:1 for normal text and 3:1 for large text (18pt+ or 14pt bold). Level AAA requires 7:1 for normal text and 4.5:1 for large text. Higher = better readability.
How do I generate a color palette?
Click the 🎭 Palette tab, pick a base color, and get 5 harmony types: Complementary (opposite), Analogous (adjacent), Triadic (3 evenly spaced), Split-Complementary, and Monochromatic (varied lightness). Click any swatch to copy its HEX.
How do I convert HEX to RGB?
Enter a HEX code like #3b82f6 in the Picker tab. RGB appears instantly. Each hex pair = one channel: 3b=59 Red, 82=130 Green, f6=246 Blue.
What are complementary colors?
Complementary colors sit opposite on the color wheel (180° apart). They create maximum contrast. Example: blue ↔ orange. Great for CTAs and accent colors against a base.
What is HSL?
HSL = Hue (0–360° on the color wheel), Saturation (0–100% from gray to vivid), Lightness (0–100% from black to white). More intuitive than RGB for adjusting shades — change lightness without shifting hue.
Is my data private?
100%. All color math runs in your browser via JavaScript. No server calls, no cookies, no tracking. Even the Share URL encodes colors in the query string — no backend involved.
Can I export palettes as CSS?
Yes! The Palette tab has an "Export CSS Variables" section. Click Copy to get all palette colors as :root { --primary: #xxx; ... } ready for your stylesheet.
How do I find the closest Tailwind CSS color?
Use the 🎯 Tailwind Finder tab. Enter any HEX color and the tool instantly shows the top 5 closest Tailwind CSS colors with class names (bg-blue-500, text-blue-500) and match percentage. Uses weighted perceptual color distance for accurate results.
Does it include all Tailwind CSS v3 colors?
Yes! All 22 color families × 11 shades = 242 colors from Tailwind CSS v3.x: slate, gray, zinc, neutral, stone, red, orange, amber, yellow, lime, green, emerald, teal, cyan, sky, blue, indigo, violet, purple, fuchsia, pink, and rose. Each with shades 50–950.
