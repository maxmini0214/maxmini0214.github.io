❓ Frequently Asked Questions

How do I convert HEX to RGB?
Enter a HEX color code like #3b82f6 and the RGB value appears instantly. Each pair of hex digits represents Red (3b=59), Green (82=130), and Blue (f6=246). In JavaScript: parseInt('3b', 16) gives 59.
What is the difference between HEX, RGB, and HSL?
HEX uses base-16 notation (#RRGGBB), RGB uses decimal 0-255 per channel, and HSL uses Hue (0-360°), Saturation (0-100%), Lightness (0-100%). HSL is often more intuitive for designers — you can adjust brightness without changing the hue.
How do I share a color with my team?
Click the 🔗 Share button to copy a URL with the color encoded. When your teammate opens the link, the color loads automatically in all formats. No account needed.
Can I use this for CSS variables?
Yes! The CSS Variable field shows ready-to-paste syntax: --color: #3b82f6; Just click 📋 and paste into your stylesheet. Works with any CSS preprocessor too.
Is my data sent to any server?
No. Everything runs 100% in your browser. No data is sent anywhere, no cookies, no tracking, no account needed. The Share URL encodes the color in the URL itself — no server involved.
How do I convert RGB to HEX in code?
JavaScript: '#' + [r,g,b].map(v => v.toString(16).padStart(2,'0')).join('')
Python: f'#{r:02x}{g:02x}{b:02x}'
CSS: rgb() and hex are interchangeable in all modern browsers.
